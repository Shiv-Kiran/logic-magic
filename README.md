# MagicLogic

MagicLogic is out of personal curiosity and confusion for math proofs and logic. It is a simple proof assistant that allows you to write proofs in a natural language style, and then checks the validity of the proofs using a simple logic engine.

Link to Website: [MagicLogic](https://mathmagiclogic.shivkiranbagathi.com/)

## Features
- Natural language proof writing
- Simple logic engine for proof checking
- Support for basic logical connectives and quantifiers
- Interactive proof construction
- Visualization of proof trees
